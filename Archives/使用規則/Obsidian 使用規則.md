# 所有被歸檔的筆記
[[MOC]]

## 標籤定義與分群
### 筆記相關
#### Status
- 📝 Note (我的筆記) | 任何來自我的想法
	- 🌱 Seedling (種子) |
	- ☀️ Sun/Incubation (培育) |
	- 🌲 Evergreen (長青樹) |

#### Source Type
- 📥 Seed Box (種子盒) | 任何來自外界的輸入資訊
	- 📄 Articles -> 網路文章
	- 💭 Thoughts -> 個人想法
	- 📚️ Books -> 實體書
	- 📮 SNS(Facebook/Twitter/Instagram)
	- 🎧️ Podcasts/音頻節目
	- 📼 YouTube
	- 🎬 網路課程
	- 📜️ PDF/研究論文
	- 🗣 聊天 -> 談話中得到的靈感


### 非筆記相關
#### Note Type
- 📝 Note (筆記)
- 📘 Map Of Content (目錄)
