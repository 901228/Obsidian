# Blender Hotkeys

## Movement
| name                   | key      |
| ---------------------- | -------- |
| move                   | G        |
| inititalize movement   | Alt + G  |
| rotate                 | R        |
| inititalize rotation   | Alt + R  |
| scale                  | S        |
| inititalize scale      | Alt + S  |
| apply to initial value | Ctrl + A |

+ 過程中，
	+ 按 x, y, z，可以使操作只在該**軸**上完成
	+ 按 Shift + x, y, z，可以使操作只在該**平面**上完成
	+ 按數字可以調整距離、角度、比例（左上角會寫出來輸入值）
	+ 按住 Shift，用較慢的速度移動
	+ 按住 Ctrl，移動半格
